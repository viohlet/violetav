PORTFOLIO

This is my first game ever, and I built it in a week as part of my capstone project, since we are free to create what we wish. I taught myself a bit of the Phaser game engine, and was able to use it on my front end. The game also has a custom Ruby on Rails API. "Microwave Wars" uses P2 Physics, so the sprites respond to collision. Also, it has a score meter and a countdown. TexturePacker was used to put the PNG files into a spritesheet and a JSON Hash formats. As of right now, the characters are taken from the "Scribblenauts" video game (property of Nintendo DS) because of time constraints. However, I will be using "Tiled" to create my own sprites soon, and incorporate animation. In addition, I plan to add more features to my game such as states, other physics, and more collision events for "extra-time" and "bonus points." By making this game, I learned that I am capable of teaching myself a new technology quite quick! I also found out that I am REALLY into games, animation, and VR.

This is a single-page app: tool for busy (and lazy) developers to build their user stories, as a first step in their creative work. In the back-end: custom Ruby on Rails API. Front end: JavaScript, Bootstrap, Handlebars, Postgres.
